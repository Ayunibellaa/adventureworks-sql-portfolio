# SQL Queries

This folder contains SQL scripts used for data analysis and business reporting.

## Analysis Included

### Exploratory Data Analysis
- Data Overview
  - Count total number of tables
  - Count total rows in main fact table
- Fact Table Validation
  - Validate data grain
  - Check duplicate records in primary key
  - Check NULL values in foreign keys
  - Check NULL values in key measures
  - Check data time range
- Dimension Validation
  - Check unmatched ProductKey
  - Check unmatched CustomerKey
- Product Exploration
  - Analyze number of products per category
  - Identify products without category
- Customer Exploration
  - Analyze minimum & maximum customer income
  - Analyze marital status distribution
  - Analyze children count distribution

### Trend Analysis
- Annual trend performance & Year-Over-Year Growth Analysis
- Monthly Performance Trend Analysis
- Cumulative Analysis (Running Total)

### Customer Analysis
- Customer Loyalty & Spending Analysis
- Income Segment vs Product Category Analysis

### Product Analysis
- Product Category Sales Performance Analysis (All Time)
- Product Category Sales Performance Analysis (Yearly)
- Top 10 Best-Selling Products (All Time)
- Top 10 Best-Selling Products (Yearly)

### Profitability Analysis
- Category Profitability Analysis (All Time)
- Category Profitability Analysis (Yearly)
- Product Profitability Analysis (All Time)
- Product Profitability Analysis (Yearly)

### Geographic Analysis
- Top Performing Country (All Time)
- Top Performing Country (Yearly)
- Best-Selling Product Category by Country (All Time)

--- 

Each SQL file contains a group of related queries focused on a specific analysis area.

Each query is designed to answer specific business questions and provide actionable insights.
